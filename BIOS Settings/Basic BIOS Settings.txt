================================
==  Intel Desktops & Laptops  ==
================================

First! Reset your BIOS to factory defaults.
首先，默认你的BIOS出厂设置，即恢复默认值

Disable all overclocking options.
关闭所有超频设置和配置

#### Note: 
Most of these options may not be present in your firmware, we recommend matching up as closely as possible but don't be too concerned if many of these options are not available in your BIOS
有些选项可能不存在你的BIOS中，或者以其它名称展示，但对于关键设置，无须但心，大部分相关。

=== Disable需关闭禁用的 ===
- Fast Boot（快速启动）
- Secure Boot（安全引导，安全启动，重要！）
- Serial/COM Port（串口）
- Parallel Port（打印口）
- VT-d (can be enabled if you set DisableIoMapper to YES)
- Intel VMD(If your BIOS has Intel VMD, you must disable it.)
- Compatibility Support Module (CSM) (Must be off in most cases, GPU errors/stalls like gIO are common when this option is enabled)
- Thunderbolt (For initial install, as Thunderbolt can cause issues if not setup correctly)
- Intel SGX
- Intel Platform Trust
- CFG Lock (MSR 0xE2 write protection) (This must be off, if you can't find the option then enable AppleXcpmCfgLock under Kernel -> Quirks. Your hack will not boot with CFG-Lock enabled)

 === Enable需打开启用 ===
- VT-x
- Above 4G Decoding （大于4G解码，重要！）
- Resizable BAR(Optional,Set config.plist-Booter-ResizeAppleGpuBars to 0 if Enabled, or -1 if Disabled.)
- Hyper-Threading
- Execute Disable Bit
- EHCI/XHCI Hand-off
- OS type: Windows 8.1/10 UEFI Mode (some motherboards may require "Other OS" instead)
- DVMT Pre-Allocated(iGPU Memory): 64MB or higher （如果提供修改，一定要64MB或者更大，重要！）
- SATA Mode: AHCI （重要！）
